FAQ

Common questions

Can you combine services into one package? +

Yes. A common combination is a Brand Identity Kit plus a Growth social media plan plus a Starter chatbot — message me with what you need and I'll quote the bundle, usually at a small discount over booking each separately.

What's the difference between the n8n and Vapi services? +

n8n builds text-based automations — chatbots and workflows that run on WhatsApp, Instagram, or your website. Vapi builds voice agents that actually answer or make phone calls. Many clients use both: a chatbot for messages and a voice agent for calls.

Do you manage social media accounts I already have, or only new ones? +

Both. If you already have pages, I'll review what's working before changing anything. If you're starting fresh, I'll set up the profiles as part of the first month.

How do payments work? +

One-time projects (design, chatbots, voice agents) are 50% upfront and 50% on delivery. Monthly plans (social media management, maintenance) are billed at the start of each month.

I'm not sure which service I need. +

That's fine — send a message describing the problem you're trying to solve, not the service you think you need. I'll tell you honestly which package fits, or if you need a custom mix.
